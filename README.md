# E-Commerce Customer Purchase Analysis 

## 📊 Project Overview
This project focuses on analyzing customer purchase behavior in an e-commerce dataset using SQL.  
The objective is to extract meaningful business insights such as revenue trends, customer demographics, and product performance.

This analysis helps understand customer purchasing patterns, identify top-performing product categories, and evaluate payment method preferences.

---

## 🛠️ Tools & Technologies
- **SQL** – Data cleaning and analysis  
- **MySQL Workbench** – Query execution  
- **Power BI** – Dashboard and data visualization  
- **Canva** – Presentation design and project showcase  


---

## 📂 Dataset Information
The dataset contains transactional customer purchase records with the following fields:

- Customer ID
- Customer Name
- Purchase Date
- Product Category
- Product Price
- Quantity
- Total Purchase Amount
- Payment Method
- Customer Age
- Gender
- Returns

---

## 🧹 Data Cleaning Process
The following data cleaning steps were performed using SQL:

- Removing duplicate records
- Standardizing column names
- Checking for missing values
- Validating total purchase calculations
- Formatting date columns

---

## 📈 Exploratory Data Analysis (SQL Queries)

The following business questions were analyzed:

1. Total Revenue Generated
2. Revenue by Product Category
3. Top-Selling Product Categories
4. Average Order Value
5. Product Return Analysis
6. Payment Method Usage Analysis

---

## 📊 Dashboard
A data visualization dashboard was created to present key insights, including:

- Total Revenue KPI
- Sales by Product Category
- Customer Demographics
- Return Rate Analysis
- Payment Method Distribution

---

## 🎯 Key Insights
Key insights derived from the analysis include:

.Books and Clothing generate the highest revenue.

.Credit Card is the most used payment method.

.Return rate is relatively high across products.

.Average Order Value (AOV) is 2721.82.
